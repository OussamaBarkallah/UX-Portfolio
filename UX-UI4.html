Redesign of Mon Atelier Écofrugale

Project Overview

During an M1 Master’s hackathon, I worked in a team on the redesign of “Mon Site Écofrugale”, a community-driven website focused on eco-friendly and frugal living. The client’s main objective was to improve the user experience and make the website easier to navigate and understand. We followed a complete UX process and delivered updated UI screens and a consistent design system.

Problem

The existing website showed usability limitations that impacted the overall experience:

  • Navigation and content structure were not intuitive
  • Key information was difficult to find quickly
  • The interface lacked consistency, reducing readability and trust

Before
After

Solution

We redesigned the website with a stronger focus on usability and clarity:

  • Improved information architecture and navigation logic
  • Clearer content hierarchy for faster scanning
  • A more consistent UI supported by a Design System
  • A refreshed homepage layout designed to guide users toward key content and actions

Impact

  • Better clarity and readability across the interface
  • Reduced navigation friction and improved content discovery
  • A more consistent and professional user experience aligned with user needs

Product Overview

The website uses a green and white color palette with subtle rose accents. Green is applied through light patterns to enhance visual identity and memorability without overwhelming the interface. The overall design follows a clean and minimalist approach, prioritizing clarity, readability, and intuitive interaction. Interactive elements are used to encourage user participation and support clear communication across the platform.
